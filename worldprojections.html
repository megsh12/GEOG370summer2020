QGIS and Projections - a GEOG 370 project

To display different projections in QGIS, you have to change the project coordinate reference system (CRS) in project properties. This can be accessed by pressing the current ESPG or ESRI number in the right corner, or 'project' then 'properties' on the top menu bar. You then select the desired CRS using either the authority ID or CRS name. This applies the CRS to the project, creating different projections.

By Meagan Harrington

North Pole Azimuthal Equidistant Projection

This is an equidistant azimuthal projection, meaning direction and distance are preserved. Size and shape are distored, particularly towards the edge of the projection. All meridians are straight, with the center point defined as the North Pole. This style of projection is used in the UN logo.

Winkel Tripel Projection

This map aims to minimise distortion of area, distance and direction by averaging the Aitoff and equirectangular projection. Distortion increases towards the edges, away from the central meridian. Polar areas are particularly exaggerated. It is used by National Geographic.

Mercator Projection

This is a conformal projection. Direction, angles and shapes are preserved - this is because the projection was originally designed for navigation. Shape is distored, particularly towards the poles. Greenland is a key example, appearing equal in size to Africa despite an actual 14-fold difference in size.

Pseudo (web) Mercator Projection

This is a varient of the Mercator projection, featured directly above, used mainly in web mapping. Angles and distances are preserved, making it a useful tool for navigation. However, size is often distorted. It differs from the standard Mercator projection as it uses spherical formulae, whereas elliptical formulae are used in the standard Mercator projection.

World Cylindrical Equal Area Projection

This is an equal area projection, though shape is often greatly distorted. This is exaggerated closer to the poles. There is great distortion of scale close to polar regions, too.

World Equidistant Conic Projection

In this conic projection, distance is preserved along meridians. Shape, angles and area are distorted, increasingly so towards the edges of the projection.

Sphere Craster Parabolic Projection

This is an equivalent, or equal-area, projection. Shape is distoreted, particularly closer to the edge of the projection due to a significant bulge. The central medidian is around half the length of the equator, creating points at each pole.
